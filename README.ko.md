# Galaxy XR Keep Awake

[English](README.md) | [한국어](README.ko.md) | [日本語](README.ja.md)

Galaxy XR Keep Awake는 Galaxy XR을 착용한 상태로 VRChat에서 수면을 더 편하게 하기 위해 만든 작은 보조 앱입니다.

Galaxy XR의 비활성 경고를 감지하고, 의도적으로 사용하는 동안 헤드셋이 꺼지지 않도록 돕습니다.

## Windows 설치기로 설치하기

1. 최신 [Releases](https://github.com/zeee2/gxr-keep-awake/releases)에서 `GalaxyXRKeepAwakeInstaller.exe`를 다운로드합니다.
2. Windows PC에서 설치기를 실행합니다.
3. 사용할 언어를 선택합니다.
4. 설치기의 Galaxy XR 준비 화면을 따라 진행합니다.
5. USB 케이블로 Galaxy XR과 PC를 연결합니다.
6. 헤드셋 안에 USB 디버깅 허용 창이 뜨면 허용을 선택합니다.
7. 설치기에서 설치 버튼을 누릅니다.
8. Galaxy XR에서 접근성 설정을 열고 Galaxy XR Keep Awake를 활성화합니다.

APK 파일이나 ADB를 직접 설치할 필요는 없습니다.

## 설치 후

접근성 설정에서 앱이 활성화된 상태를 유지해주세요.

설정이 끝난 뒤에는 앱을 백그라운드에 두면 됩니다. 꺼짐 방지 기능을 사용하는 동안에는 앱을 강제 종료하지 말아주세요.

## 주의

이 앱은 Samsung, Google, VRChat, SteamVR, Virtual Desktop의 공식 앱이 아닙니다.

Galaxy XR은 OLED 패널을 사용합니다. 장시간 사용하거나 정적인 화면을 반복해서 표시하면 번인이 발생할 수 있습니다. 이 앱은 본인 책임하에 사용해주세요. 번인이나 디스플레이 손상에 대해 개발자는 책임지지 않습니다.

## 개발 비화

앱을 제작하면서, Samsung이 OS를 수정하는 과정에서 접근성 권한을 매우 강하게 제한해둔 것을 확인했습니다.

따라서 이 릴리스는 접근성 설정에서 앱을 활성화할 수 있도록 호환성 우회 방식을 사용합니다. 2026-06-22 기준으로 정상 동작하지만, 향후 소프트웨어 업데이트로 막힐 수 있습니다.
