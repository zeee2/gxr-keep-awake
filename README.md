# Galaxy XR Keep Awake

Galaxy XR Keep Awake is a small companion app made to make sleeping in VRChat more convenient while wearing Galaxy XR.

It watches for the Galaxy XR inactivity warning and keeps the headset active when needed.

## Features

- Works in the background after setup
- Helps prevent unwanted headset sleep
- Optional success notification
- English, Korean, and Japanese support
- No root required No internet access

## Setup

1. Install the APK on your Galaxy XR.
2. Open the app.
3. Enable the accessibility permission.
4. Allow notifications if you want confirmation messages.

After setup, you can leave the app in the background.  
*Please Do Not Close APP.*

## Notes

This app is not an official Samsung, Google, VRChat, SteamVR, or Virtual Desktop app and anyting.
